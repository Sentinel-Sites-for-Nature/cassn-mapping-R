# _reference

Canonical spatial registry of all CA-SSN sites. This is the single source of truth for site coordinates and organization metadata — downstream projects read from here rather than maintaining their own copies.

## What this project does

- Maintains `data_raw/CASSN_all_sites.csv`, the user-edited master site table
- Converts it to a spatial GeoPackage (`data/CASSN_all_sites.gpkg`) for use in mapping
- Produces a static reference map of all 109 sites colored by organization

## Data

### Input (user-maintained)

`data_raw/CASSN_all_sites.csv` — edit this file to add, remove, or update sites.

| Column | Description |
|--------|-------------|
| `organization` | CDFW, UCNRS, CSU, TNC, or Pepperwood |
| `campus` | University campus affiliation (UCNRS/CSU) or blank |
| `site_name` | Full formal site name |
| `latitude` | WGS 84 decimal degrees |
| `longitude` | WGS 84 decimal degrees |

Current site counts: CDFW 39 | UCNRS 40 | CSU 26 | TNC 3 | Pepperwood 1 = **109 total**

### Output (generated)

`data/CASSN_all_sites.gpkg` — WGS 84 point layer; regenerated by `convert_all_sites_to_spatial.R`. Do not edit manually.

## Scripts

| Script | Role |
|--------|------|
| `00_setup.R` | Install and load required packages into project-local `.Rlib` |
| `convert_all_sites_to_spatial.R` | Read CSV → validate coordinates → write GeoPackage |
| `map_all_sites.R` | Static PNG map colored by organization + site-count table |

## Outputs

`outputs/maps/all_sites.png` — 11×8.5" at 300 dpi, colored by organization.

## Typical run order

1. Edit `data_raw/CASSN_all_sites.csv` if site data has changed
2. Run `scripts/00_setup.R`
3. Run `scripts/convert_all_sites_to_spatial.R`
4. Run `scripts/map_all_sites.R`
